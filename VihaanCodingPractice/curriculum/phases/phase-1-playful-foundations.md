# Phase 1: Playful Foundations

## Goal

Build comfort with code as instructions. Focus on events, movement, sounds, coordinates, and repeat blocks through playful projects in Scratch.

## Concepts

- Sequence
- Events
- Motion
- Looks and sound
- Coordinates
- Repeat loops

## Tools

- Scratch only

## Week-by-Week Plan

### Week 1: Meet Scratch
- Session A: Explore the stage, sprites, blocks, and green flag
- Session B: Make a character move and say hello
- Project: Animated hello story
- Child can say: "The green flag starts my project."

### Week 2: Steps in Order
- Session A: Put actions in sequence
- Session B: Add costume changes and sounds
- Project: Dance party animation
- Helpful variation: Morning routine animation

### Week 3: Repeat
- Session A: Use repeat to make actions happen again
- Session B: Compare one-time steps versus repeat steps
- Project: Bouncing or dancing character
- Child can say: "Repeat saves time."

### Week 4: Move on a Map
- Session A: Introduce x and y positions as locations on the stage
- Session B: Guide a sprite through simple paths
- Project: Mini maze
- Helpful variation: Walk through a school map

### Week 5: Click to Play
- Session A: Start actions with clicks
- Session B: Count taps informally with visible changes
- Project: Clicker game
- Helpful variation: Water-the-plant helper

### Week 6: Make It Feel Alive
- Session A: Add sounds, messages, and costume switches
- Session B: Give characters personalities and reactions
- Project: Pet simulator starter
- Helpful variation: Homework reminder character

### Week 7: Remix Week
- Session A: Pick a favorite earlier project and improve it
- Session B: Add one new block type and one design change
- Project: Favorite remix

### Week 8: Showcase
- Session A: Plan a small project using events and repeat
- Session B: Finish and share
- Showcase options:
  - Pet game
  - Maze challenge
  - Morning routine helper
  - Dance story

## Recommended Projects

Games:
- Maze game
- Clicker game
- Pet play scene

Helpful tools:
- Morning routine helper
- Homework reminder scene

## End-of-Phase Check

The child can:
- Start projects with an event
- Use repeat in a meaningful way
- Explain that coordinates place things on the screen
- Build a small project without copying every single step
