# Phase 3: Gentle Bridge to Typed Code

## Goal

Show that typed code is another way to express ideas the child already knows from blocks. Keep typing light and focus on remixing and tiny edits.

## Concepts

- Editing existing code safely
- Numbers and colors
- Variables in text form
- Simple if statements
- Input and output
- Comparing blocks to code

## Tools

- Scratch for comparison and comfort
- p5.js or Code.org App Lab for beginner typed code

## Week-by-Week Plan

### Week 17: Blocks and Code Match-Up
- Session A: Compare a Scratch motion block to a small code example
- Session B: Change one number and observe the result
- Project: Move-a-shape remix

### Week 18: Colors and Art
- Session A: Change colors, size, or text in a starter sketch
- Session B: Personalize with the child's name
- Project: Name art generator

### Week 19: Bounce and Motion
- Session A: Edit speed values in a bouncing ball project
- Session B: Change what happens at the edge
- Project: Bouncing ball

### Week 20: Click and Draw
- Session A: Use mouse interaction in a drawing toy
- Session B: Change brush behavior or color choices
- Project: Drawing toy

### Week 21: Useful Numbers
- Session A: Edit a simple counter or tracker
- Session B: Add a reset button or message
- Project: Points tracker
- Helpful variation: Sticker chart helper

### Week 22: Random Choices
- Session A: Build or remix a random picker
- Session B: Change the choices to foods, games, or chores
- Project: Random choice picker

### Week 23: Tiny If Statements
- Session A: Show a basic "if score is high, show message"
- Session B: Modify the message and rule
- Project: Countdown helper or mini calculator

### Week 24: Showcase
- Session A: Choose a favorite typed-code starter and plan changes
- Session B: Finish and share
- Showcase options:
  - Name art generator
  - Drawing toy
  - Countdown helper
  - Random choice picker

## Recommended Projects

Games:
- Bouncing ball
- Drawing toy

Helpful tools:
- Name art generator
- Points tracker
- Random choice picker
- Countdown helper
- Mini calculator

## End-of-Phase Check

The child can:
- Change a small piece of typed code on purpose
- Explain what one line or value changes
- Connect blocks they know to typed-code ideas
- Stay calm while debugging small text mistakes
