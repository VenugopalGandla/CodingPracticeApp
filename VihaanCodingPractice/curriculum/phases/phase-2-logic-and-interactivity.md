# Phase 2: Logic and Interactivity

## Goal

Connect game rules to real-world app behavior. Introduce variables, score, timers, collisions, and choice blocks in Scratch.

## Concepts

- Conditionals
- Variables
- Score
- Timer
- Collision detection
- Simple debugging

## Tools

- Scratch

## Week-by-Week Plan

### Week 9: Score as a Memory Box
- Session A: Create a score variable
- Session B: Change score when something good happens
- Project: Catching game

### Week 10: Choice Blocks
- Session A: Use if blocks to react to game events
- Session B: Add winning or losing rules
- Project: Treasure catch game
- Helpful variation: Chore tracker with points

### Week 11: Timers
- Session A: Introduce countdowns or race-against-time play
- Session B: Add a simple end screen
- Project: Reaction game
- Helpful variation: Speed clean-up challenge

### Week 12: Questions and Answers
- Session A: Use ask/answer or quiz-like interactions
- Session B: Add feedback for correct and incorrect answers
- Project: Quiz game
- Helpful variation: Classroom quiz helper

### Week 13: Feelings and Tracking
- Session A: Build buttons that change states
- Session B: Use variables to save simple choices
- Project: Feelings check-in tool

### Week 14: Practice App
- Session A: Build a basic math or reading helper
- Session B: Add score and retry options
- Project: Math practice app or reading log

### Week 15: Debugging Week
- Session A: Look at a broken project and fix one bug at a time
- Session B: Add labels and better sprite names
- Project: Bug-fix remix

### Week 16: Showcase
- Session A: Plan a project with score, choice, and one goal
- Session B: Finish and share
- Showcase options:
  - Catching game
  - Reaction game
  - Chore tracker
  - Reading log

## Recommended Projects

Games:
- Catching game
- Quiz game
- Reaction game

Helpful tools:
- Chore tracker
- Reading log
- Feelings check-in
- Classroom quiz helper

## End-of-Phase Check

The child can:
- Use a variable as a memory box
- Make something happen only if a condition is true
- Build a project with score or timer
- Find one simple bug by testing a small change
