# Program Overview

## Summary

This program teaches coding fundamentals through a mix of games and useful tools. It starts with block coding in Scratch, then slowly introduces typed code once the child already understands the main ideas.

The child learns:
- Sequence: putting steps in order
- Events: code starts when something happens
- Repeat: making the computer do a step again and again
- Choice: if this happens, do that
- Variables: memory boxes that hold changing information
- Debugging: noticing what went wrong and fixing one thing at a time

## Year Structure

- 4 phases
- 8 weeks per phase
- 2 sessions each week
- 30 to 40 minutes per session

Each phase ends with a showcase project.

## Session Structure

Each session follows the same rhythm:
- Warm-up: 5 minutes
- New idea: 5 to 10 minutes
- Build time: 15 to 20 minutes
- Share and test: 5 minutes

## Success Markers

The child is progressing well when they can:
- Predict what a project will do before clicking run
- Explain a block or line of code in their own words
- Find a bug with a hint
- Change one rule, art choice, or number on purpose
- Finish small projects and feel proud sharing them

## Teaching Style

- Use simple language first, official vocabulary second
- Keep activities playful
- Let the child choose themes often
- Use lots of remixing before starting from a blank page
- Praise noticing and testing, not just correctness

## Recommended Monthly Themes

- Month 1: Animals
- Month 2: Sports
- Month 3: Art and music
- Month 4: Space
- Month 5: School helpers
- Month 6: Family routines
- Month 7: Nature and weather
- Month 8: Invent your own theme
