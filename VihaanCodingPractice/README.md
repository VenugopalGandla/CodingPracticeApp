# Coding Fundamentals for a 3rd Grader

This workspace contains a full school-year coding curriculum for a 3rd grader who is learning to build games and helpful real-life tools.

The curriculum now also includes a kid-friendly website entry point:
- [index.html](/Users/nethravathin/Documents/New%20project/VihaanCodingPractice/index.html)

GitHub Pages deployment is prepared with:

- [404.html](/Users/nethravathin/Documents/New%20project/VihaanCodingPractice/404.html)

The curriculum is designed for:
- 2 sessions per week
- 30 to 40 minutes per session
- Laptop + browser
- Light adult help
- Low typing at the beginning

## Program Goals

By the end of the program, the child should be able to:
- Explain and use sequence, repeat, choice, events, variables, and debugging
- Build at least 4 games and 4 useful mini-tools
- Modify projects with growing independence
- Create one mostly original final project
- See coding as a way to make both fun things and helpful things

## Tool Path

- Phase 1: Scratch
- Phase 2: Scratch
- Phase 3: Scratch plus beginner typed code in p5.js or Code.org App Lab
- Phase 4: Child chooses Scratch or beginner typed code depending on confidence

## Notes

- The plan is intentionally gentle and repeatable.
- Remixing counts as real learning.
- A finished small project is better than an unfinished big one.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. Make sure your default branch is `main`.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `GitHub Actions`.
5. Push to `main` or run the `Deploy static site to GitHub Pages` workflow manually.

After deployment, the site will be published at your GitHub Pages URL.
