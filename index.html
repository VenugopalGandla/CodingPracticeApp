Weekly Rhythm

Small sessions. Big progress.

Session A

  • Warm-up: remember what you built last time
  • Learn one fresh idea
  • Build or remix a mini-project
  • Explain one thing your code does

Session B

  • Predict what will happen before you run it
  • Add one challenge or new rule
  • Turn it into a game or a useful helper
  • Test and share what to improve next

For Grown-Ups

How to help without taking over

Helpful things to do

  • Open the website or project
  • Read tricky instructions aloud
  • Ask what the child wants the code to do
  • Encourage testing after small changes

Great coaching questions

  • What do you think will happen?
  • Which part starts the project?
  • What changed after you tested it?
  • What is one tiny thing to try next?
