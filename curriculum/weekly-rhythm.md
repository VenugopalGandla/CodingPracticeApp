# Weekly Rhythm

Use this rhythm in every session so the child knows what to expect.

## Session A

- Warm-up: Ask, "What did we build last time?"
- New idea: Show one concept only
- Build: Make or remix a small project
- Share: Ask the child to explain one block or code change

## Session B

- Warm-up: Predict what the project will do before running it
- New idea: Add one challenge or variation
- Build: Improve the project or turn it into a useful tool
- Share: Test it together and note one thing to improve next time

## Session Rules

- Keep projects small
- Stop while the child still feels successful
- Choose one new concept per session
- Reuse old concepts often
- Let the child personalize art, names, sounds, and colors

## Weekly Success Check

By the end of each week, the child should be able to do at least one of these:
- Change a sprite, button, or background
- Adjust a score or timer
- Add a sound or message
- Fix one bug with a hint
- Explain how the project starts
