# Phase 4: Create and Invent

## Goal

Move from guided projects to child-owned creation. Reinforce planning, debugging, polish, and sharing.

## Concepts

- Planning before building
- Breaking a big idea into parts
- Testing in small steps
- Improving based on feedback
- Choosing the right tool for the job

## Tools

- Scratch or beginner typed code
- Child chooses based on confidence and interest

## Week-by-Week Plan

### Week 25: Choose a Big Idea
- Session A: Pick game or tool ideas
- Session B: Draw a simple plan with screens, characters, or buttons
- Project: Idea map

### Week 26: Build the First Working Version
- Session A: Make the project start and do one important thing
- Session B: Test and fix the first bugs
- Project: First playable or first usable version

### Week 27: Add Rules
- Session A: Add score, progress, or choice logic
- Session B: Improve visuals or labels

### Week 28: Add User Help
- Session A: Add instructions, restart button, or end screen
- Session B: Let someone else test it

### Week 29: Build a Helpful Tool
- Session A: Choose a real-life problem to help with
- Session B: Make a tool that solves one small part of it
- Project ideas:
  - Study helper
  - Pet care helper
  - Allowance tracker
  - Kindness challenge app

### Week 30: Improve from Feedback
- Session A: Ask what worked and what was confusing
- Session B: Make 2 to 3 small improvements

### Week 31: Final Polish
- Session A: Clean up names, labels, and screens
- Session B: Practice explaining the project

### Week 32: Final Showcase
- Session A: Final test and presentation prep
- Session B: Share portfolio

## Final Project Options

Games:
- Platform-lite game
- Adventure game
- Animal rescue game
- School quest game

Helpful tools:
- Study helper
- Healthy habit tracker
- Pet care helper
- Allowance tracker
- Kindness challenge app

## End-of-Program Check

The child can:
- Plan a small project with a clear purpose
- Finish a project and explain how it works
- Improve the project after feedback
- Choose whether a game or tool is the better fit for an idea
