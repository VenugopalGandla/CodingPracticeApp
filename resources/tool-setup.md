# Tool Setup

## Scratch

- Website: [Scratch](https://scratch.mit.edu/)
- Best use in this program: Phases 1 and 2, plus optional Phase 4 final projects
- Recommended setup:
  - Create a free account if you want to save and share online
  - Start with remixing starter projects
  - Use full-screen workspace when possible

## p5.js

- Website: [p5.js Web Editor](https://editor.p5js.org/)
- Best use in this program: Phase 3 typed-code starters
- Recommended setup:
  - Adult helps open a starter sketch
  - Child mainly edits numbers, colors, names, and short messages at first

## Code.org App Lab

- Website: [Code.org](https://code.org/)
- Best use in this program: Beginner apps with buttons, labels, and simple logic
- Recommended setup:
  - Useful if the child likes tool-style projects more than drawing-style projects
  - Adult may need to help with navigation

## Choosing Between p5.js and App Lab

- Pick p5.js if the child likes art, motion, and playful visuals
- Pick App Lab if the child likes buttons, screens, and helper apps
- It is fine to use only one typed-code tool in Phase 3
